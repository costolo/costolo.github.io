SQL Cheat Sheet

SQL! Love it? Hate it? Don't understand it? Here's a handy cheat sheet to help keep the confusion (and excessive Googling) at bay.

SELECT

SELECT field_name1, field_name2

FROM table_name;

WHERE

SELECT field_name

FROM table_name

WHERE field_name (> <=) argument;

OR

WHERE field_name LIKE "%string%";

ORDER BY

SELECT field_name

FROM table_name

ORDER BY field_name ASC/DESC;

INNER JOIN

SELECT table1_name.field_name, table2_name.field_name

FROM table1

INNER JOIN table2

ON table1.field_name = table2.field_name;
